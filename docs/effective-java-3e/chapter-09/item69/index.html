[아이템 69] 예외는 진짜 예외 상황에서만 사용하라.

예외는 반드시 예외 상황에서만 사용해야한다. 일반적인 제어 흐름용으로 사용하면 안 됩니다.

    int i = 0;
    while (true) {
        index[i++].doSomething();
    }

    ..
}

코드가 장황하고 직관적이지 않습니다. 성능도 일반적인 제어 흐름보다 느립니다.
